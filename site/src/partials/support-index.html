Support Nova

Nova is an open-source project under active development by a single maintainer. Here’s how you can help.

Financial support

Direct financial support helps cover ongoing costs (AI API usage, cloud, services) and gives the maintainer room to keep pushing the project forward.

  • GitHub Sponsors — one-time or monthly support, accepts most payment methods worldwide. github.com/sponsors/unitcraft (if profile is active — otherwise being set up).
  • Boosty — payment platform for Russian individuals. Currently being set up; this section will be updated when ready.
  • Corporate sponsorship — opens after the 0.1 release; meanwhile, write to hello@nv-lang.org for preliminary discussions.

Code & documentation

Pull requests, issues, and code review are equally welcome. Areas that need help most:

  • Standard library — std/STATUS.md tracks what needs work.
  • Testing on real code — port a small piece of your work to Nova on paper, point out where it breaks down.
  • Documentation translations — RU↔EN polish, other languages welcome.
  • PL theory review — especially effect-system formalisation. Issues open at github.com/nv-lang/nova/issues.

Spread the word

If Nova’s ideas resonate — tell a colleague, share an article, write your own take. Word-of-mouth from credible practitioners is worth more than any marketing.

Questions about supporting in some other way? Email hello@nv-lang.org.
