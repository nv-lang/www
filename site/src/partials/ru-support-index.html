Поддержать Nova

Nova — open-source проект в активной разработке одним мейнтейнером. Вот как можно помочь.

Финансовая поддержка

Прямая финансовая поддержка покрывает текущие расходы (AI API, облако, сервисы) и даёт мейнтейнеру возможность двигать проект дальше.

  • GitHub Sponsors — разовая или ежемесячная поддержка, принимает большинство способов оплаты по всему миру. github.com/sponsors/unitcraft (если профиль активен — иначе настраивается).
  • Boosty — платформа для физических лиц в РФ. Настраивается; этот раздел обновится, когда будет готово.
  • Корпоративное спонсорство — откроется после релиза 0.1; пока что напишите на hello@nv-lang.org для предварительного обсуждения.

Код & документация

Pull requests, issues и code review — одинаково ценны. Области, где помощь нужна больше всего:

  • Стандартная библиотека — std/STATUS.md ведёт список того, что нужно доделать.
  • Тестирование на реальном коде — перенесите небольшой кусок своей работы в Nova на бумаге, ткните в дыры.
  • Перевод документации — полировка RU↔EN, другие языки приветствуются.
  • Ревью PL-теории — особенно формализация effect-системы. Issues открыты на github.com/nv-lang/nova/issues.

Расскажите

Если идеи Nova резонируют — расскажите коллеге, поделитесь статьёй, напишите собственный взгляд. Слово от живого практика стоит больше любого маркетинга.

Вопросы про другие способы поддержки? Пишите на hello@nv-lang.org.
